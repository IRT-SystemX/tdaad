43  
44 From a multivariate timeseries, form a similarity matrix and apply a TDA procedure to produce

46  
47 Read more in the :ref: `User Guide <persistence_diagrams>`.
48  
49 Parameters:
50 tda_max_dim : int, default=2
51 The maximum dimension of the topological feature extraction.

55

177 class TopologicalEmbedding(LocalPipeline):
178 """Topological embedding for multiple time series.
179  
180 Slices time series into smaller time series windows, forms an affinity matrix on each window
181 and applies a Rips procedure to produce persistence diagrams for each affinity
182 matrix. Then uses Atol [ref:Atol] on each dimension through the
183 gudhi.representation.Archipelago representation to produce topological vectorization.
184  
185 Read more in the :ref:`User Guide <topological_embedding>`.
186  
187 Parameters
188 ----------
189 window_size : int, default=40
190 Size of the sliding window algorithm to extract subsequences as input to named_pipeline.
191 step : int, default=5
192 Size of the sliding window steps between each window.
193 n_centers_by_dim : int, default=5
194 The number of centroids to generate by dimension for vectorizing topological features.
195 The resulting embedding will have total dimension =< tda_max_dim * n_centers_by_dim.
196 The resulting embedding dimension might be smaller because of the KMeans algorithm in the Archipelago step.
197 tda_max_dim : int, default=2
198 The maximum dimension of the topological feature extraction.
199 min_persistence : float, default=0.0
200 The minimum persistence value to take into account (the persistence value will be strictly
201 greater than min_persistence).
202 remapping : str, default="simple"
203 The method to use to compute final results out of the sliding window algorithm.
204 By default, "simple" means a `simple_index_remapping` procedure is applied, meaning that each window produces
205 an anomaly score for its last timestamp. Other options are "vote" where `majority_vote_index_remapping` when
206 a score for a certain timestamp is computed by voting over all windows that include that timestamp,
207 and "none" where no remapping is applied, meaning that output indices will be tuples of size `window_size`.
208  
209 Examples
210 ----------
211 >>> n_timestamps = 100
212 >>> n_sensors = 5
213 >>> timestamps = pd.to_datetime('2024-01-01', utc=True) + pd.Timedelta(1, 'h') * np.arange(n_timestamps)
214 >>> X = pd.DataFrame(np.random.random(size=(n_timestamps, n_sensors)), index=timestamps)
215 >>> TopologicalEmbedding(n_centers_by_dim=2, tda_max_dim=2).fit_transform(X)
216 """
